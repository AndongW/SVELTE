R Scripts and Objectives
1. SimData.R: Generates multistage survival data with censoring and partition labels. 
2. WithinPartitionAlg_RMST.R: Implements the single-partition SVELTE / recursive forest RMST estimator. 
3. IPCW_RMST.R: Implements a one-partition IPCW RMST policy evaluator.
4. Example_WithinPartitionAlg_RMST.R: Minimal example script showing how to simulate data and fit the recursive RMST estimator.
5. SVELTE_IPCW_comp.R: Comparison workflow utilities for SVELTE versus IPCW, including replicate studies and approximate truth via uncensored simulation.
