R Scripts and Objectives
1. SimData.R: Generates multistage survival data with censoring and partition labels. 
2. WithinPartitionAlg_RMST.R: Implements the single-partition SVELTE RMST policy value estimator with the "Concatenation/Extension" augmentation scheme.
3. WithinPartitionAlg_RMST_1StubExt.R: Implements the single-partition SVELTE RMST policy value estimator with the "1-stub extension for all" augmentation scheme.
4. Example_WithinPartitionAlg_RMST.R: Minimal example script showing how to simulate data and fit the recursive RMST estimator.
5. IPCW_RMST.R: Implements a one-partition IPCW RMST policy value estimator.
6. SVELTE_IPCW_comp.R: Comparison workflow utilities for SVELTE versus IPCW, including replicate studies and approximate truth via uncensored simulation.
